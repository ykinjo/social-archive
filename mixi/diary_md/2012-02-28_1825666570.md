# [mixi] 東上

**作成日:** 2012-02-28

長崎で満9年を迎えましたが、3月末に退職し、新年度は東京で迎えることになりました。3月下旬に引越します。丸3年、東京と長崎を行ったり来たりしましたが、晴れてゲゲゲタウンに落ち着きます。



地味に就職活動はしてたのですが、成果なく、当面無職です。

救いの神はいるもので、先週、とある研究会に顔を出したら4月からの非常勤講師にドタキャンが出たら連絡するといってくださる都内の大学の先生がいて、履歴書を預かってもらいました。キャンセルが出るよう祈ってます
。



部屋も研究室も引越しまでに片付くのか、途方にくれてますが、学生の手伝いを確保したので、人海戦術でなんとかなることを祈ってます。



冷蔵庫とか、ベッドとか、洗濯機とか、大きいものの処分も頭が痛かったのですが、この春に卒業して一人暮らしをはじめる子がまとめて引き取ってくれることになり、こちらは心配がなくなりました。
