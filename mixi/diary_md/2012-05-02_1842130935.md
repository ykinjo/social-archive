# [mixi] 引越しから約1ヶ月

**作成日:** 2012-05-02

引越して約1ヶ月。近くのスーパーの特売のローテーションなどもわかってきました
。



3年前から生活してた土地ではあるのですが、車があると少し活動範囲が変わって、発見もたくさんあります。都心にはほとんど出てません
。



・桜がすごく多い。多摩川に近いから？

・街路樹に花みずきが多い。今花ざかりですが、今まで住んだ場所で、花が咲く街路樹って（つつじは別として）見たことなかった。

・運転してて何が怖いかって自転車。長崎はほとんどいなかったので。子ども乗せてる人とか、高齢者の自転車とかキョーフです。雨の日でも傘さして乗ってる人多いし。

・世田谷あたりの住宅街は迷宮。

・探偵ナイトスクープが金曜の夜になくて落ち着かない。

・スーパーで見かける食材で珍しいのは生シラス。あと、チカという魚を初めて見た。（まだ食べたことない）。

・食材は全体的に少し高いけど、広告の品が安いので組み合わせると、長崎よりベラボーに高いってことはない。

・なぜかヨーグルトの値段が高い。平均148円くらいで買ってたけど、198円くらいなのがイラつく。特売で138円とかはあるんですけどねえ。

・生花が高い。100円とか200円で買ってたのが倍くらい。大きい額ではないけど、なかなか手が出ない。
