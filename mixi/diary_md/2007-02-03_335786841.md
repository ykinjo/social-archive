# [mixi] POTSUNEN 『○ -maru-』フライヤーの謎かけ？

**作成日:** 2007-02-03

先月DVDが届いてから観まくってるわけですが、色々謎があるので、昨年3月舞台を観に行った時もらったフライヤーをしげしげと眺めてみました。



ラーメンズ関係のフライヤーはえらく凝ってます。



『○ -maru-』の時は2枚のフライヤーをもらいました。

1枚はDVDのジャケットに使われてる点描の写真（？）、もう一枚は挨拶状のような文字が主体のフライヤー。



文字主体のフライヤーは、挨拶と言葉遊びが取り混ぜてあり、数字もたくさん出てきます。八百は800とかわざわざ書いてたり、文中に現れる数字が変わるごとに意味ありげな「／」も出てくるし、数字に何か意味があるのだろうかとふと思いました。



出てくる数字は順に



0, 1, 2, 3, 4, 4.5, 5, 7, 12, 365, 9, 800, 2006, 0



もしかしたら0になるのかもと思って、いろいろ計算してみたら、ちゃんと0になりました。



私の妄想じゃなくて、これって明らかに狙ってるんですよねえ。



1年近く気がつかなかったし、blogとかの劇評たくさん読んでるけど、誰もそんなこと書いてなかったなあ。
